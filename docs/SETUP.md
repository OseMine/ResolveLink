# Setup Guide

## Prerequisites

| Requirement | Version | Notes |
|-------------|---------|-------|
| **Node.js** | 18+ | Download from [nodejs.org](https://nodejs.org) |
| **Python** | 3.10+ | Required for Resolve scripting bridge |
| **DaVinci Resolve Studio** | 21+ | Free version does not support scripting API |
| **Adobe After Effects** | 2022+ | Any version with CEP support |
| **REAPER** | 6+ | *Optional* — for audio post-production workflow |
| **OS** | Windows 10/11, macOS 12+, or Linux | |

---

## Quick Install (DaVinci Resolve Console)

1. Download `install.lua` from the [latest release](https://github.com/OseMine/ResolveLink/releases)
2. Open DaVinci Resolve
3. Go to **Workspace > Console** (or press Alt+F9)
4. Paste the contents of `install.lua` and press Enter

The installer will:
1. Detect your OS and check for Git + Node.js
2. Download the correct release version automatically
3. Run the full setup (deps, build, extensions, .env)
4. Deploy Resolve scripts and start the server

## Install REAPER Scripts

### Via ReaPack (Recommended)

1. In REAPER, go to **Options > Show REAPER resource path**
2. Open `Scripts/ReaPack/Packages/`
3. Go to **ReaPack > Edit repositories**
4. Add this URL:
   ```
   https://raw.githubusercontent.com/OseMine/ResolveLink/main/index.xml
   ```
5. Click **OK**, then **ReaPack > Synchronize packages**
6. Go to **ReaPack > Browse packages**
7. Search for "ResolveLink" and install

### Available Scripts

| Script | Description |
|--------|-------------|
| **ResolveLink Panel** | Unified control panel (all-in-one, no dependencies) |
| **ResolveLink Callback** | Background worker for auto-import |
| **ResolveLink Send to Resolve** | Render and import audio to DaVinci |
| **ResolveLink Update Project** | Sync REAPER items to match DaVinci timeline |
| **ResolveLink Logger** | Toggleable console logger for debugging |

### Using the Panel

1. In REAPER, go to **Actions > Show action list**
2. Click **Load** and select `ResolveLink-panel.lua`
3. Click **Run** (or assign to a toolbar button)
4. The panel window opens with all controls

**Panel Controls:**
- **Callback Toggle** — ON/OFF to start/stop background job polling
- **Send Render to Resolve** — Renders audio and imports to DaVinci
- **Update Project from Resolve** — Syncs REAPER items to match DaVinci timeline
- **Status Display** — Shows current state (idle, active, error)
- **Log Console** — Scrollable log of recent activity

## Manual Installation

### 1. Install Dependencies

```bash
npm install
cd src && npm install && cd ..
```

### 2. Build Frontend

```bash
cd src && npm run build && cd ..
```

### 3. Deploy Extensions

```powershell
.\scripts\deploy-extension.ps1
.\scripts\deploy-resolve-script.ps1
```

### 4. Configure

Copy `.env.example` to `.env` and set your paths, or run the setup wizard:

```powershell
.\scripts\setup.ps1 -Force
```

### 5. Start

```powershell
.\scripts\start.ps1
```

## Configuration (.env)

All paths are configured via `.env` (auto-generated by setup wizard):

| Variable | Default | Description |
|----------|---------|-------------|
| `PORT` | `3030` | Server port |
| `HOST` | `127.0.0.1` | Bind address (localhost only for security) |
| `EXPORT_DIR` | `./exports` | Render output directory |
| `TEMP_DIR` | `./temp` | Generated scripts directory |
| `PYTHON_PATH` | `python` | Python executable path |
| `AE_PATH_WIN` | *(auto-detected)* | After Effects path (Windows) |
| `AE_PATH_MAC` | *(auto-detected)* | After Effects path (macOS) |
| `REAPER_PATH_WIN` | *(auto-detected)* | REAPER path (Windows) |
| `REAPER_PATH_MAC` | *(auto-detected)* | REAPER path (macOS) |
| `RESOLVE_SCRIPTING_PATH` | *(auto-detected)* | Resolve scripting modules path |
| `WATCHER_ENABLED` | `true` | Enable file watcher |
| `WATCH_FOLDERS` | *(none)* | Extra folders to watch (comma-separated) |
| `LOG_LEVEL` | `info` | Log level: debug, info, warn, error |

## CEP Extension

### Automatic Install

```powershell
.\scripts\setup.ps1
# or just the extension:
.\scripts\deploy-extension.ps1
```

### Manual Install

1. Enable CEP debug mode:
```powershell
reg add "HKCU\Software\Adobe\CSXS.11" /v PlayerDebugMode /t REG_SZ /d 1 /f
```

2. Copy extension files:
```powershell
$cepDir = "$env:APPDATA\Adobe\CEP\extensions\com.resolvelink.panel"
New-Item -ItemType Directory -Force -Path $cepDir
Copy-Item -Recurse -Force "extension\client" "$cepDir\client"
Copy-Item -Recurse -Force "extension\CSXS"  "$cepDir\CSXS"
Copy-Item -Recurse -Force "extension\host"  "$cepDir\host"
```

3. Restart After Effects → **Window > Extensions > ResAE Link**

## Troubleshooting

### "Bridge disconnected"
1. Ensure server is running: `.\scripts\start.ps1`
2. Check port 3030: `netstat -ano | findstr 3030`

### "After Effects not found"
1. Check `aerender.exe` exists at the configured path
2. Run `.\scripts\setup.ps1 -Force` to re-detect

### Plugin not showing in Resolve
1. Must use DaVinci Resolve **Studio** (not free)
2. Restart Resolve after installing
3. Check **Workspace > Console** for errors

### CEP Extension not showing
1. Enable debug mode (see above)
2. Restart After Effects
3. Check **Window > Extensions**

### Python bridge hangs
- Python 3.10–3.13 may hang calling `scriptapp("Resolve")`
- Use Python 3.14 if available, or ensure the correct Python version is in `.env`

### Port in use
Change `PORT` in `.env` and restart.

### "REAPER not found"
1. Check that REAPER is installed and the path is correct in `.env` (`REAPER_PATH_WIN` / `REAPER_PATH_MAC`)
2. Run `.\scripts\setup.ps1 -Force` to re-detect paths
3. Default paths: `C:\Program Files\REAPER (x64)` (Windows), `/Applications/REAPER.app` (macOS)

### REAPER Panel not loading
1. Ensure the panel script is deployed to REAPER's Scripts directory
2. In REAPER, go to **Actions > Show action list** and verify the script is listed
3. If using ReaPack, ensure packages are synchronized
4. Check the REAPER console for error messages

### REAPER callback not importing
1. Ensure the panel's callback toggle is ON (green button)
2. Check that `exports/reaper-jobs/` directory exists
3. Verify the server is running and writing job files
4. Check the panel's log console for error messages

### Audio files not detected by file watcher
1. Ensure `WATCHER_ENABLED=true` in `.env`
2. Audio formats watched: `.wav`, `.flac`, `.aiff`, `.ogg`, `.mp3`
3. Rendered files must be in the `exports/` directory or a watched folder

## Uninstalling

```powershell
# Stop server
.\scripts\stop.ps1

# Remove CEP extension
Remove-Item -Recurse "$env:APPDATA\Adobe\CEP\extensions\com.resolvelink.panel"

# Remove Resolve script
Remove-Item "$env:APPDATA\Blackmagic Design\DaVinci Resolve\Support\Fusion\Scripts\Utility\send-to-ae.py"
Remove-Item "$env:APPDATA\Blackmagic Design\DaVinci Resolve\Support\Fusion\Scripts\Utility\send-to-reaper.py"
Remove-Item "$env:APPDATA\Blackmagic Design\DaVinci Resolve\Support\Fusion\Scripts\Utility\ResolveLink.lua"

# Remove REAPER scripts
Remove-Item "$env:APPDATA\REAPER\Scripts\ResolveLink-*"

# Clear generated files
.\scripts\clear.ps1 all

# Delete the project folder
Remove-Item -Recurse "X:\coding\AE-Link"
```
